= Vert.x examples

image:https://img.shields.io/badge/vert.x-4.5.4-purple.svg[link="https://vertx.io"]

Bunch of examples used to illustrate features of Eclispe Vert.x. To keep it simple, each class file has a main function to run the corresponding example.

== Building

To launch your tests:
```
./mvnw clean test
```

To package your application:
```
./mvnw clean package
```

To run your application:
```
./mvnw clean compile exec:java
```
